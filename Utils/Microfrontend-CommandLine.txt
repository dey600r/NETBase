MICROFRONTEND BUILDTIME
-------------------------------

- ANGULAR --- CREATE CONTAINER
    ng new app-container --create-application=false

- ANGULAR --- CREATE LIBRARY
    ng generate library security-lib

- ANGULAR --- CREATE app
    ng g application app-module

- ANGULAR --- BUILD PACK
    npm install -D ng-packagr
    ng build security-lib --configuration production
    npm pack

MICROFRONTEND RUNTIME
----------------------------------
- ANGULAR --- INSTALAR WEBPACK MODULE FEDERATION
    npm i @angular-architects/module-federation@19.0.3
    ng add @angular-architects/module-federation --port 4200 --project app-module

- INIT EMPTY VUE NODE PROJECT
    npm init -y
    npm i @babel/core @babel/preset-env babel-loader webpack webpack-cli webpack-dev-server @vue/compiler-sfc html-webpack-plugin style-loader css-loader vue vue-loader

- INIT VUE PROJECT
    npm create vue@latest
